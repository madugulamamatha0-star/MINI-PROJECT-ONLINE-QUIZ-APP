# MINI-PROJECT-ONLINE-QUIZ-APP

##Online Quiz App (Java Console Mini Project):

A simple and interactive console-based quiz application built in Java.Users answer multiple-choice questions, and the app calculates the final score.

##Objective:

1.Build a Java mini-project that:

2.Runs completely on the console

3.Displays MCQ questions with options

4.Accepts user answers

5.Evaluates correctness

##Features

✨ Console-based clean UI

✨ Stores questions using a List

✨ Uses Question Class

✨ MCQ options (1–4)

 ✨ Final result output

✨ Easy to expand with more questions
✨ Score tracking
